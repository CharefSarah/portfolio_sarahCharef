A propos

Shield-bearers est un jeu de type rpg qui vous plonge dans le monde fantastique de Roznot. Vous débutez l'aventure dans le petit village de Myrefall. Vous devrez alors choisir quel héros incarner,
pour délivrer le monde des forces du chevalier noir Void.

Voir le projet

L'univers

Le jeu comporte plus d’une vingtaine de personnages dont huit jouables. La carte quant à elle contient plus d’une quarantaine de niveaux. De nombreux mini-jeux sont également accessible
le tout rythmé par une ambiance sonore de qualités.
